# J's Digital — Electronics & Tech

A sleek, dark-themed electronics storefront built with vanilla HTML, CSS, and JavaScript.

## 🚀 Live Demo

> After enabling GitHub Pages, your site will be at:
> `https://<your-username>.github.io/<repo-name>/`

## ✨ Features

- Browse products by category (Audio, Computing, Mobile, Accessories, Gaming)
- Search and filter (Sale, New Arrivals, Featured)
- Product detail modals with specs
- Shopping cart with quantity controls
- Checkout flow with order summary
- User registration & login (stored in localStorage)
- Add custom products via form

## 📁 Project Structure

```
index.html   ← Entire app (HTML + CSS + JS, self-contained)
README.md    ← This file
```

## 🌐 Deploy to GitHub Pages

1. **Create a new GitHub repository** at https://github.com/new
2. **Upload `index.html`** and this README to the repo root
3. Go to **Settings → Pages**
4. Under *Source*, select **Deploy from a branch**
5. Choose **`main`** branch and **`/ (root)`** folder
6. Click **Save** — your site will be live in ~60 seconds

## 🛠️ Local Development

Just open `index.html` directly in any browser — no server needed.

## 📝 Notes

- All data is stored in the browser's localStorage (no backend)
- Products reset on page refresh (default seed data is hardcoded)
- Fonts are loaded from Google Fonts (requires internet connection)
